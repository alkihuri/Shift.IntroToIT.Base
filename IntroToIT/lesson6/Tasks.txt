Перед вами представлены задачи по Python, 

Структура проекта : LessonX - папка урока, где Х номер уркоа
                        TaskY - папка задания, где Y номер задания
                            Task.py - Файл с неправильно работающим кодом который вам нужно исправить


каждая из которых демонстрирует распространенные ошибки, 
с которыми сталкиваются начинающие разработчики. 
Ваша задача — изучить предоставленный код, определить, что в нем неправильно, и улучшить его!

Пример: 
# Неправильная функция
def fahrenheit_to_celsius(f):
    return (f - 32) / 9/5

#правильная функция 
def fahrenheit_to_celsius(f):
    return (f - 32) * 5/9
